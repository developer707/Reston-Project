To run:

open git bash and run git clone https://github.com/developer707/Reston-Project.git
or directly download the file.

Open note Pad++ and drag the Item Inventory Folder into it so that the entire folder will be ther. the run the index.html file in Chrome.

Or open the  ItemInventory folder in any other text editor And open the index file in a Chrome Browser.

Also use the Chrome extension CORs to allow the service call to go through and receive data from the Walmart Api.
